## 今日收获

按键音的format是在/data01/jyxi/8838_new/QSSI.16/frameworks/base/media/jni/soundpool/Sound.cpp 中decode写死的
```
  *audioFormat = AUDIO_FORMAT_PCM_16_BIT;  // default format for audio codecs.
```
因此导致一个问题就是如果pcm解析出来的是32bit 他仍然给audiotrack传16. 这就导致了audioflinger不会根据输出流配置（配置的是16bit）做resample.那么给audiohal送的pcm真实的是32bit。但是又告诉audiohal我给你的是16bit 导致播放是杂音
因此我这边直接把源文件给改了，改成16bit
